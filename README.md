# 🏠 AI Smart Home Automation System using ESP32

An AI-integrated Smart Home Automation System built using ESP32, Blynk IoT, and Sinric Pro for intelligent wireless appliance control through mobile application and Google Assistant voice commands.

This project demonstrates the practical implementation of IoT, cloud communication, and embedded systems for modern smart home applications. The system allows users to control household appliances such as bulbs and fans remotely using smartphones or voice assistants over WiFi.

## 🚀 Features

* 📱 Mobile control using Blynk IoT
* 🎙 Google Assistant voice control
* ☁️ Sinric Pro cloud integration
* ⚡ Real-time relay switching
* 🌐 WiFi-based wireless automation
* 🏠 Smart home appliance control
* 🔌 ESP32-based embedded automation system
* 📡 Remote access through internet

## 🧰 Components Used

* ESP32 Development Board
* 3-Channel Relay Module
* Bulbs
* Fan
* Jumper Wires
* Breadboard
* WiFi Network

## 🛠 Software Used

* Arduino IDE
* Blynk IoT
* Sinric Pro
* Google Home

## ⚙️ Working Principle

The ESP32 connects to WiFi and communicates with Blynk IoT and Sinric Pro cloud platforms. Users can control appliances through the Blynk mobile app or using Google Assistant voice commands. The ESP32 processes the received commands and switches connected appliances ON or OFF using relay modules.
